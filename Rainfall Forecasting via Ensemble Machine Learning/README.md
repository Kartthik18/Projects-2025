🌧️ Rainfall Forecasting via Ensemble Machine Learning
📌 Overview

This project focuses on predicting rainfall using an ensemble of machine learning models trained on the Australian Weather Dataset (~140K+ records).
The objective is to design a robust, end-to-end pipeline and improve predictive performance through model comparison and ensembling.

🚀 Key Highlights
Built a complete ML pipeline from preprocessing to evaluation
Achieved 86.20% accuracy (best model) and 84.15% ensemble accuracy
Compared multiple models using Accuracy, F1-score, ROC-AUC, and PR-AUC
Led a team of 3, guiding model development and optimization

🧠 Models Implemented
Logistic Regression
Random Forest
XGBoost
Feedforward Neural Network (FFNN)
Weighted Soft Voting Ensemble
